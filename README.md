# SmartWallet - Expense Tracker

Personal finance app to track expenses and budgets

*Part of WILP course project*

## Features

- Add expenses with description, amount, and date
- View all expenses with smart date display (Today/Yesterday)
- Delete expenses
- View total expenses
- Added chart features
- Added goal setting feature
- Added Export functionality.
